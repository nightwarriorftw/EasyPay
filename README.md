## Development Environment

 **Clone the Repo**
 ```BASH
 git clone EasyPay
 ```
 **Intall the Dependencies**
 ```BASH
 npm install
 ```
 
**Run the applicaton**
```BASH
expo start
```
*To run on browser user `-w` flag and on mobile device use `-a` flag after the above command*
